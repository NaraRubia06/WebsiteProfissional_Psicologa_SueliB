# Website Profissional para Psicologa Sueli Balena.
O objetivo deste projeto é criar um website dedicado à psicologia que sirva como vitrine digital e ferramenta educacional. Os principais objetivos são promover o trabalho profissional, divulgar serviços, especializações e métodos de comunicação, além de oferecer conteúdo informativo sobre saúde mental, bem-estar e autoconsciência. Dessa forma, a plataforma visa conectar a psicóloga com a comunidade, ampliar o acesso a informações confiáveis ​​na área da psicologia e proporcionar-lhes acesso a aconselhamento e informação.
